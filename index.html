Say Cheese! - QR Code Edition

Minimal javascript library for integrating a webcam and snapshots into your app.

Taking pictures

Say Cheese exposes a simple, event-based API for taking snapshots from your webcam:

{% highlight javascript %} $(function() { var sayQr = new SayQRCode('#container-element', { snapshots: true }); sayQr.on('start', function() { this.takeSnapshot(); }); sayQr.on('snapshot', function(snapshot) { // a snapshot has been taken, do something with it :) }); sayQr.setQROptions( function(successData) { if(successData != "error decoding QR Code") { // do something with the output of the QR Code decoding. } }, function(errorData) { // do something with the error messages produced by decoding the QR Code. }); sayQr.start(); sayQr.startQrRead(1000); // try to decode the image every 1 second. }); {% endhighlight %}

Compatibility

Works for the following browsers:

  • Google Chrome
  • Opera
  • Firefox
